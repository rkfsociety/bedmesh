# 📏 Bed Mesh Visualizer

Профессиональный инструмент для анализа карты стола 3D-принтера на базе **Klipper**.

## ✨ Особенности
* **🚀 Умный импорт**: Авто-парсинг файла `printer_mutable.cfg` (JSON и текстовый форматы).
* **📊 Side-by-Side View**: Компактное отображение 3D и 2D графиков в один ряд.
* **🔧 Мастер выравнивания**: Пошаговые инструкции для настройки винтов (обороты) или валов Z (миллиметры).
* **🖱️ Полный интерактив**: Вращайте и изучайте рельеф в реальном времени.

## ⬇️ Скачать

- **Windows (PyQt6)**: скачайте файл `Bed.Mesh.Visualizer.exe` из раздела **Releases** этого репозитория.

## 🔄 Автообновления (Windows / PyQt6)

В приложении есть **тихая проверка обновлений** при запуске:

- Внизу **правой панели** показывается текущая версия (например `v0.152-win`)
- Если доступна новая версия, вместо версии будет **«Требуется обновление (vX → vY)»** и кнопка **«Обновить»**
- Нажатие **«Обновить»** скачивает новый `.exe` из GitHub Releases и перезапускает приложение

## ▶️ Запуск из исходников (Windows / PyQt6)

### Требования
- Python 3
- Зависимости из `bedmesh/win/pyqt6/requirements.txt`

### Запуск

Из папки `win/pyqt6`:

```powershell
py -3 main.py
```

## 🏷️ Релизы (как публиковать новую версию)

Для Windows (PyQt6) релиз публикуется по тегу:

- Тег формата **`vX.YYY-win`** (например `v0.152-win`)
- После пуша тега GitHub Actions собирает `.exe` и прикрепляет его к релизу
- Имя файла в релизе всегда: **`Bed.Mesh.Visualizer.exe`**

Пример:

```powershell
git tag v0.153-win
git push origin v0.153-win
```
